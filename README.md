# Multimodal RAG Application

A powerful Retrieval-Augmented Generation app that works with multiple data sources including PDFs, images, YouTube videos, and research papers.

## Features

- 📄 PDF and document processing
- 🖼️ Image analysis and understanding
- 🎥 YouTube video transcription
- 📚 ArXiv paper search
- 🔍 Semantic search with ChromaDB
- 🤖 Multiple LLM integrations (Groq, Gemini)

## Usage

1. Add your API keys in the sidebar
2. Upload documents or enter queries
3. Get AI-powered responses with source citations
